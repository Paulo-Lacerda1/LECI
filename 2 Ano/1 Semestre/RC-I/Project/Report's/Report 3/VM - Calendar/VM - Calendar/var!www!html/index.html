Como Jogar o Jogo "Guess the Number" (Cliente)

O jogo "Guess the Number" é um jogo em que o cliente tenta adivinhar um número gerado pelo servidor.

  1. Assegure-se de que o servidor do jogo está em execução.
  2. Tenha o IP e a porta do servidor de jogo. Ex: 203.157.191.250:5005
  3. Utilize o cliente para ligar ao servidor:
    • Python: python3 client.py
  4. Introduza o seu palpite e siga as instruções do servidor.

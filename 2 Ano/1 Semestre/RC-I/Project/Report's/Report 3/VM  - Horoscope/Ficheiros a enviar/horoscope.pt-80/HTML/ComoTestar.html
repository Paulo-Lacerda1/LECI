Como testar os Pings entre PCs ?

O que é um ping?

O comando ping é usado para testar a ligação entre dois dispositivos numa rede. Existem diferentes tipos de pacotes que circulam na rede (ARP,ICMP,TCP,UDP...).

Como usar o comando Ping ?

  1. Abre a Consola de um VPC
  2. Escreve o comando: ping [endereço IP ou nome do host].
  3. Clica Enter para executar o comando.
  4. Verifica o sucesso do ping, caso contrário envia mensagem privada para o Discord de um professor da cadeira ;)

Ping através do nome:

São os mesmos passos de um ping normal, só muda o comando:

ping NOMEdoPC.horoscope.pt

Como Jogar "Guess the number"

O Jogo:

"Guess the number" é um jogo que precisa de dois PC's. Um para o servidor onde vai gerar um número aleatório entre 0-1000 e outro para o cliente que vai tentar adivinhar esse número. O jogo acaba quando acertar em cheio no número!

Arquitetura Cliente/Servidor:

  • O Servidor gera um número aleatório dentro de um intervalo configurado (0-1000).
  • O Cliente é utilizado pelo jogador para enviar palpites ao servidor.
  • O Servidor responde com dicas, indicando se o jogador deve tentar um número maior ou menor.
  • O Servidor também regista o número de tentativas de cada jogador (identificado pelo seu IP).
  • Quando o jogador acerta, o servidor envia uma mensagem de vitória e pergunta se quer jogar de novo.
  • O registo das pontuações também é guardado num ficheiro leaderboard.txt

Como Jogar?

  1. Iniciar o servidor com o comando sudo python3 server.py (substituir a palavra server pelo nome do ficheiro do servidor)
  2. Após o servidor inicar, para conectar ao servidor usa-se o comando sudo python3 cliente.py
  3. Depois da ligação estar estabelecida basta começar a jogar.
  4. Para sair, no final do jogo basta não continuar ou fazer CTRL + C

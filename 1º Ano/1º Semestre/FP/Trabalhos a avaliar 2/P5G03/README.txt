Caso o código não funcione pode ser que seja necessário instalar o módulo requests.

Para o instalar:
-Abre o terminal;
-Digita pip install requests
-Click ENTER

Para verificar se foi instalado:
-Digita: python3 -c "import requests; print(requests.__version__)"
Clica ENTER
Se não houver erros e for apresentada a versão do módulo, este está instalado.
